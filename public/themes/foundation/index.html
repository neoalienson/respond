Flexible Plugins

Respond features built-in support for forms, galleries, and menus allowing your users to easily manage their site without touching any code. Quickly adapt our built-in controls to match your Foundation-based designs.

Visual Editor

Our visual editor allows you users to edit the content in place, so you don't have to spend hours on training to get new users up to speed. Just login, select a page, and begin editing.

Powerful Tools

Respond allows you to quickly edit galleries, forms and menus from the main menu. So, you can quickly add a photo, collect new feedback, or add a new menu item with ease.

Really, Really Fast

Respond sites are built with static HTML so they run really fast. All the plugins are pre-rendered, so you never have to wait for the server.
